## Install dependencies
```
pip install usellm
pip intall flask
```

## Run the app
```
flask run
```

## Steps to use the APP
1. Open Codespace
2. Install dependencies
3. Run the app
4. Click "Open in browser"
5. Provide link of article. Example: https://www.deccanherald.com/international/world-news-politics/erdogan-seeks-third-decade-of-rule-in-turkish-runoff-1222607.html
6. Click generate
